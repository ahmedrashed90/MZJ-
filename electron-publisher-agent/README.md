# MZJ Publisher Agent

برنامج Electron مبدئي لقراءة أجندة النشر من فولدرات الجهاز وحفظ مهام النشر في Firebase داخل collection باسم `publishing_jobs`.

## شكل الفولدر المدعوم

```text
Agenda/2026/6/
├── 17-6/
│   ├── بوست/
│   ├── ريل/
│   ├── ستوري/
│   └── كابشن/
│       └── كابشن.docx
└── تم النشر/
```

القواعد:
- `بوست`: كل الصور داخل الفولدر = بوست Carousel واحد.
- `ريل`: كل فيديو = Reel مستقل.
- `ستوري`: كل صورة/فيديو = Story مستقلة بدون كابشن.
- `كابشن/كابشن.docx`: النص الكامل المستخدم مع البوست والريل.

## التشغيل

```bash
cd electron-publisher-agent
npm install
npm start
```

بعد حفظ المهام، افتح السيستم على صفحة **جدولة النشر المحلي** لعرض المهام.

> تنفيذ النشر الفعلي على المنصات سيستخدم الحسابات المربوطة من صفحة **ربط المنصات** في السيستم.
